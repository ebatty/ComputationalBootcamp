Creating & Reshaping Arrays#

import numpy as np

Creating Arrays#

There are many ways to intialize arrays. Let’s start with the basics

  1. Initialize with numbers in closed brackets (python lists)

x = np.array([4, 6, 1])

x

array([4, 6, 1])

x = np.array([[4, 6, 1], [3, 4, 5]])

print(x)
print(x.shape)

[[4 6 1]
 [3 4 5]]
(2, 3)

  1. Initialize with zeros or ones

x = np.zeros(shape = (2, 3))
x

array([[0., 0., 0.],
       [0., 0., 0.]])

x = np.ones(shape = (5, 1))
x

array([[1.],
       [1.],
       [1.],
       [1.],
       [1.]])

  1. Initialize with evenly spaced numbers

Given interval and step size:

x = np.arange(start = 0, stop = 10, step = 2)

print(x)

[0 2 4 6 8]

Given interval and specified number of points:

x = np.linspace(start = 0, stop = 10, num = 50)

Concatenating arrays#

x = np.array([[3, 2], [1, 4]])

print(x)

[[3 2]
 [1 4]]

y = np.concatenate((x, x), axis = 1)

y

array([[3, 2, 3, 2],
       [1, 4, 1, 4]])

y = np.concatenate((x, x), axis = 0)

y

array([[3, 2],
       [1, 4],
       [3, 2],
       [1, 4]])

Reshaping arrays#

x = np.array([[1, 2, 3], [4, 5, 6]])

print(x)

[[1 2 3]
 [4 5 6]]

y = np.reshape(x, newshape = (6, 1))

print(y)

[[1]
 [2]
 [3]
 [4]
 [5]
 [6]]

y = x.reshape((6, 1))

print(y)

[[1]
 [2]
 [3]
 [4]
 [5]
 [6]]

y = x.reshape((3, 2))

print(y)

[[1 2]
 [3 4]
 [5 6]]

x = np.zeros((5, 7, 10))

y = x.reshape((5, -1))

y.shape

(5, 70)
