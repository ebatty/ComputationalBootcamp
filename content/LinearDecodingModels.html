def plot_observed_vs_predicted(n_1, angle, angle_hat, w_0, w_1):
  """ Plot observed vs predicted data

  Args:
    n_1 (ndarray): observed activity values of neuron 1
    angle (ndarray): observed angle values
    angle_hat (ndarray): predicted angle values
    w_0 (float): value of w_0 parameter
    w_1 (float): value of w_1 parameter
  """
  fig, ax = plt.subplots()
  ax.scatter(n_1, angle, label='Observed')  # our data scatter plot
  ax.plot(n_1, angle_hat, color='r', label='Fit')  # our estimated model

  ax.set(
      # title=fr"$\hat{{\theta}}$ = {theta_hat:0.2f}, MSE = {np.mean((y - y_hat)**2):.2f}",
      xlabel='Neuron 1 activity',
      ylabel='Angle'
  )
  ax.legend()

Step 1: Define a mapping (model) from input to output#

In order to fit a model to data, we first need to define our model. In this case, we will use linear decoding.

(1)#\[\begin{align} a &: \text{orientation of grating}\\ n_1 &: \text{spike count of neuron 1}\\ a &= w_0 + w_1n_1\\ \end{align}\]

The weights \(w_0\) and \(w_1\) are what we need to learn from data.

You probably tried to by eye to get the red line as close as possible to the observed data. Specifically we want to minimize the difference between prediction and observed data along the y axis (the actual angle vs the predicted angle).

We need a quantitative measure of how bad the model is, called a cost or loss function. We will then be able to find parameters that minimize that measure.

The equation for computing the estimated angle for a single data point \(i\) is:

(3)#\[\begin{equation} \hat{a}_{i}= w_0 + w_1 n_{1, i} \end{equation}\]

For each data point, we can compute the different between the estimate and actual angle: \(\hat{a}_{i} - a\). We will square this difference so higher is worse, no matter whether the prediction is higher or lower than the real value. We then average this term over all the data points to get our loss function, mean squared error:

(4)#\[\begin{equation} \text{Mean squared error} = \frac{1}{N}\sum_{i=1}^{N}\left(a_{i}-\hat{a}_i\right)^{2} \end{equation}\]

where N is the number of data points (in this case, the number of trials)

Step 3: Find parameters that minimize the loss function#

We want to find the values of \(w_0\) and \(w_1\) that minimize mean squared error on our data set. We’ll ignore the details of algorithms that do this for now. Just know that there are smart ways to automatically find the minimum of functions and figure out what parameters lead to that minimum, but they’re not always perfect.

For now, we’ll use the Python library sklearn, which will fit the model for us: https://scikit-learn.org/stable/modules/generated/sklearn.linear_model.LinearRegression.html
